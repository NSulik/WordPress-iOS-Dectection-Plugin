== Description ==
A plugin that determins whether the user is running iOS 6, iOS 5 or any other operating system. 
Depending on the result it inserts a smart banner, simulates a smart banner or does nothing respectively.

== Installation ==

1. Upload `iOS-SB-Inserter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin (iOS SmartBanner Inserter) through the 'Plugins' menu in WordPress.
3. Make sure your template uses the 'wb_head' hook.
